# Tarifle Sosyal Medya Content Calendar

> Web launch playbook §2 ek (`docs/FUTURE_PLANS.md`). 4 haftalık
> şablon + ilk 28 post taslağı. Brand voice rehberi:
> `docs/SOCIAL_MEDIA_BRAND_VOICE.md`.

## Haftalık Tema Şablonu

| Gün | Tema | Format |
|---|---|---|
| Pazartesi | Yeni tarif tanıtım | Mod R görsel + 1 cümle hook + ingredient teaser + CTA |
| Salı | Blog cross-post | Yazıdan 1 alıntı + görsel + "Devamı blog'da" CTA |
| Çarşamba | Hızlı ipucu / mit çürütme | Editöryal görsel + 3-5 cümle bilim + tarif örneği |
| Perşembe | Mevsimsel tarif öneri | Mevsim sembolü + 2-3 tarif kartı + koleksiyon link |
| Cuma | Cuisine spotlight | Ülke bayrağı + 4-6 tarif grid + mutfak rehberi blog |
| Cumartesi | Kullanıcı UGC / hikaye (gelecek) | Repost + krediler + topluluk vurgusu |
| Pazar | Haftalık özet / koleksiyon | Top 3 görüntülenen + featured shelf + newsletter signup |

## Hesap Setup Checklist

Launch günü öncesi her platform için:
- [ ] Profil avatarı (logo PNG 400x400)
- [ ] Banner / cover photo (Instagram 1080x1920 story, Twitter
      1500x500, Pinterest 800x450)
- [ ] Bio metni TR + EN (160 char Twitter, 150 IG)
- [ ] Link in Bio: tarifle.app
- [ ] İlk 5-7 post backlog hazır (zamanlanmış)
- [ ] Notification ayarları (DM, mention, comment)

## Brand Voice Uyumu

Her post'tan önce kontrol:
1. Em-dash var mı? (yasak, AGENTS.md)
2. Hashtag 5-10 arası, mix TR + EN + niche
3. CTA var ve link doğru
4. Görsel Mod R aesthetic uyumlu
5. TOV samimi + somut + otoriter değil
